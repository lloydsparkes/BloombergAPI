MessageProperties
This class represents properties of a message that are not part of the message contents, for example the CorrelationID, or timestamp.
TestUtil
Provides utilities for unit testing BLPAPI applications.

This includes deserializing services from XML, creating test events, and appending/formatting messages in these events.

Interfaces
  InterfaceDescription
IMessageFormatter
Used to populate/format a message. It supports writing once only to each field. Attempting to set an already set element will fail.

Behavior is undefined if:

  • A message formatted with FormatMessageXml(String) is further formatted.
  • A message formatted with a Set method or an Append method is further formatted using FormatMessageXml(String).
